# Once Upon a Time #1 PR — Landing Page Brief

## Cliente
Once Upon a Time #1 PR

## Tipo de negocio
Centro de empoderamiento para la mujer, con dos marcas/programas propios bajo el mismo techo:
- Pioneer Latina
- Chic Divas Treasure

## Objetivo del sitio
Landing informativa + contacto. NO es ecommerce, NO pide donaciones, NO tiene formulario de aplicación formal. El trabajo es presentar la organización, su misión y sus dos marcas/programas, con una vía de contacto clara.

## Importante
- No inventar datos duros (año de fundación, cifras, alianzas reales, número de miembros). Todo copy factual se marca como borrador hasta que el cliente lo confirme o mande contenido real.
- No cambiar logos ni los colores de marca ya establecidos.
- No reciclar layout/estilo de otros proyectos del estudio (el-senor-pique, linkeapr, Terramar) — debe sentirse construido específicamente para esta marca.
- Seguir `CLAUDEwebdesign copy.md` (ver copia en Downloads del cliente) al pie de la letra: cero purple/gradient-text/glassmorphism/stat-blocks genéricos/emoji en headings/"Why choose us".

## Stack
- Sitio estático: HTML5, CSS3, JavaScript vanilla (no WordPress — a diferencia de el-senor-pique, este va a GitHub → Vercel).
- Higgsfield MCP: recrear los logos en alta calidad manteniendo colores exactos, y generar animaciones ("cobrar vida") de esos assets.
- Hyperframes: componer/animar esas piezas en secuencias pulidas (hero, transiciones) dentro de la página.
- Deploy: GitHub (MediaWaveGlobalLLC) → Vercel.

## Estilo visual
Elegante, dorado sobre negro, cuento de hadas moderno, empoderamiento femenino, premium — no infantil, no genérico corporativo.

## Colores (preliminar — se confirman en el plan de diseño)
- Negro: #0A0A0A
- Dorado: #C9A227
- Dorado claro / highlight: #E8C766
- Rosa empolvado (de Chic Divas Treasure): #D98A9E
- Marfil: #FAF6EE

## Tipografía
Poppins pedido explícitamente por el cliente. Se define pairing con una display face de más carácter para headlines en el plan de diseño (Poppins solo como sistema completo se ve genérico).

## Marcas en la página
1. **Once Upon a Time #1 PR** — marca/organización principal → hero + misión.
2. **Pioneer Latina** — programa propio, sección dedicada.
3. **Chic Divas Treasure** — programa propio, sección dedicada.

## Referencia estructural (NO visual)
Screenshot de un sitio de "Centro para Emprendedores" (azul/naranja) — se usa solo como referencia de qué tipos de secciones tener (hero, franja de marcas, misión/servicios, eventos, tarjetas "cómo ayudamos", contacto/footer). El estilo visual final es 100% dorado/negro de Once Upon a Time, no ese azul/naranja.

## Secciones (borrador de IA, sujeto al plan de diseño)
1. Header / nav
2. Hero — thesis visual + animación del logo
3. Nuestras marcas — Once Upon a Time, Pioneer Latina, Chic Divas Treasure
4. Misión / qué hacemos
5. Eventos / próximamente
6. Cómo te ayudamos (bloques alternados imagen+texto)
7. Contacto
8. Footer

## Contenido/copy
No hay copy real todavía — el estudio redacta un borrador inicial (aspiracional, sin cifras inventadas) para ajustar junto con el cliente antes de dar por final.

## Regla principal
El resultado no puede sentirse "vibe coded". Diseño con paleta real de los logos, tipografía con carácter, composición no genérica, un elemento de firma memorable.
